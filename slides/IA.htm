Content surfacing #1 - what is it?

  • The IA is the skeleton
  • Each page must have a single parent but can be connected to many other related pages / sections
  • Content is modular and is 'surfaced' throughout the website
  • Specific content and sub-sections will be LCF-themed

Content surfacing #2 - how does it work?

  • Related links
  • Feeds (RSS, Showtime)
  • Shared keywords
  • Grouping content using taxonomies
  • Search and filters

Example #1
Student profile

Key

  • A - Video player (can be inserted into different contexts)
  • B - Next/previous page navigation
  • C - Manual related links (these links can be visual, show a snippet of text etc)
  • D - Keyword related links / tagging (automatically pulled in when a profile is tagged correctly)
  • E - Showtime feed (live content, updated when Elly updates her Showtime profile)

Course page
